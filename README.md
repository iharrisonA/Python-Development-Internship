# Python-Development-Internship
Command-line applications built during a Python Development Internship, covering task management, arithmetic operations, and security tooling using Python's standard library.

🐍 Python Development Internship:
This repository contains Python application projects completed as part of my Python Development Internship. Each project focuses on building a practical, real-world command-line tool using core Python concepts such as functions, modules, data structures, file handling, and user interaction.

📂 Projects
1. ✅ To-Do List App
A feature-rich task management application that allows users to add, edit, complete, delete, search, and filter tasks. Supports priorities, categories, due dates, overdue detection, and a statistics dashboard. All tasks are saved persistently to a local JSON file.
2. 🧮 Simple Calculator
A command-line calculator supporting 10 arithmetic operations including addition, subtraction, multiplication, division, power, modulo, floor division, square root, percentage, and absolute value. Features chained calculations and a full session history.
3. 🔐 Password Generator
A secure password generator that lets users specify length and complexity using 6 presets (Basic, Strong, Max, PIN, Memorable, Custom). Generates cryptographically secure passwords using Python's secrets module, with built-in strength analysis and entropy scoring.
